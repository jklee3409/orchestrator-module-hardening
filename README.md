
# Dacha (다챠)

**개인 간 모바일 데이터를 안전하고 효율적으로 거래할 수 있는 플랫폼, Dacha**

---

## 소개

Dacha는 남은 데이터를 판매하고, 부족한 데이터를 충전할 수 있는 **모바일 데이터 거래 플랫폼**입니다. 사용자 간 안전하고 투명한 거래를 돕기 위해 경매 방식, 시세 제공, 자체 결제 시스템(Pay) 등을 지원합니다.

---

## 이 리포지토리는

* **다챠 클라이언트 서비스의 백엔드 소스코드**를 포함하고 있으며,
* 데이터 거래, 결제, 인증/인가 등 주요 기능들을 구현합니다.

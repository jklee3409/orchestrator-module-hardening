# pay 진행 기록

## 목적

`pay` 도메인의 AI agent 변경 이력을 추적한다.

## 갱신 규칙

- 새 기록은 `기록` 최상단에 추가한다.
- 각 entry는 간결하게 유지한다.
- 변경 파일, 검증, 남은 위험을 포함한다.
- 관련 없는 도메인 변경은 기록하지 않는다.

## 기록 템플릿

```md
### YYYY-MM-DD - <짧은 제목>
- 범위:
- 파일:
- 변경:
- 검증:
- 위험:
```
